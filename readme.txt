== Installation ==

The plugin is simple to install:

1. Download `default-categories.zip`
1. Unzip
1. Upload `default-categories` directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin
1. Give yourself a pat on the back
